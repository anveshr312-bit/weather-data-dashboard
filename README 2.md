# Weather Data Dashboard

A premium, glassmorphism-styled weather dashboard built with Vanilla JavaScript, HTML5, and CSS3.

## Features

- **Real-time Weather Data**: Fetches data from the [Open-Meteo API](https://open-meteo.com/).
- **Glassmorphism Design**: Modern UI with frosted glass effects, deep gradients, and smooth transitions.
- **Search Functionality**: Search for any city worldwide.
- **Responsive**: Works on desktop and mobile devices.
- **Error Handling**: Graceful error messages for invalid cities or network issues.

## How to Run

1.  Simply open `index.html` in any modern web browser.
2.  No build step or server required.

## Technologies

-   HTML5
-   CSS3 (Variables, Flexbox, Grid, Animations)
-   Vanilla JavaScript (ES6+, async/await, fetch)
